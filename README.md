# Nmap Network Scanning Assignment

## Student Information

**Name:** Abhishek Nangare

## Objective

The objective of this assignment is to install Nmap, identify the local IP range, perform a TCP SYN scan on the local network, identify active hosts and open ports, analyze the detected services, and understand potential security risks associated with exposed network services.

## Tools Used

* Nmap 7.99
* Windows Command Prompt
* Windows 11
* Wi-Fi (Mobile Hotspot)

## Target Network

10.77.196.0/24

## Scan Command

```bash
nmap -sS 10.77.196.0/24
```

## Files Included

* Report.docx
* hotspot_scan.txt
* Screenshots

## Outcome

The scan successfully identified active hosts on the local network and detected several open TCP ports. The detected services were analyzed, and possible security risks were documented along with security recommendations.
